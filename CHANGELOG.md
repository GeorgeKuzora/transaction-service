# Change Log

## 0.1.0

### Новое

- Добавлены базовые абстракции для сервиса:

- Добавлен пакет тестов TransactionService. Пакет тестов включает юнит и интеграционные тесты.
- Создан файл gitlab-ci.yml. Настоен пайплайн для линтера и тестов.
